| Kullanıcı | Doğru Terim |
|-----------|-------------|
| %30 imar | TAKS = 0.30 |
| emsal | KAKS (Kat Alan Katsayısı) |
| gabari | Yükseklik sınırı |
| köşe başı | Köşe parsel |
| yola terk | Terk alanı |
| ifrazlı | İfraz (parsel bölme) |
| tevhid | Tevhit (parsel birleştirme) |
| iskan | Yapı kullanma izin belgesi |
| ruhsat | Yapı ruhsatı |
| bağımsız bölüm | Daire/dükkan/ofis gibi bağımsız ünite |
| milton | Milyon |
| buçuk | 0.5 |
| 3,5 oda | 3+1 |

| bodrum+4 | İmar: bodrum+4 normal kat |

## Alan Terimleri
- **Brüt Alan**: Duvarlar dahil toplam alan
- **Net Alan**: Kullanılabilir iç alan (duvarlar hariç)
- **ARSA için**: Sadece tek alan değeri (brüt/net ayrımı YOK)

## Oda Sayısı Formatı
- 1+1, 2+1, 3+1, 4+1, 5+2 vb.
- Dubleks: 4+1 Dubleks
- Tripleks: 5+2 Tripleks
