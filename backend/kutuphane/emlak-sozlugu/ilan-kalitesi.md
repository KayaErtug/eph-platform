# İlan Kalite ve Duygu Analizi

## İlan Tarzı → Kalite Sinyali
| İlan Tarzı | Yorum |
|-----------|-------|
| Çok ünlem (!!!) | baskılı satış |
| Sürekli büyük harf | agresif pazarlama |
| Çok kısa açıklama | düşük kalite |
| Detaylı teknik veri | profesyonel |
| Aşırı emoji | düşük güven |
| Net m² ve brüt m² ayrı | kaliteli ilan |
| Tapu türü belirtilmiş | şeffaf ilan |

## Emlakçı Kalite Skoru
```json
{
  "agentScore": 0-100,
  "fakeListingRisk": "LOW/MEDIUM/HIGH",
  "listingQuality": "LOW/MEDIUM/HIGH"
}
```

## Fiyat Gerçeklik Analizi
Şu faktörler değerlendirilmeli:
- Bölge ortalaması
- m² başına fiyat
- Bina yaşı etkisi
- Kat etkisi (üst katlar genelde pahalı)
- Cephe etkisi (güney cephe +)
- Ulaşım etkisi
