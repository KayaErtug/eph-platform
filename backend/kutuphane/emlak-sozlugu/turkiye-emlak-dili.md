# Türkiye Emlak Dili - Özel Terimler

## Yapı / İnşaat Dili
| Terim | Anlam |
|-------|-------|
| 3 kat imarlı | 3 katlı yapı yapılabilir |
| zemin artı 4 | bodrum+zemin+4 normal kat |
| kat karşılığı | arsa karşılığı daire |
| arsa paylı | hisseli arsa tapulu daire |
| yola terk | yol için bırakılan alan |
| köşe parsel | iki yola cepheli arsa |
| emsal harici | emsale sayılmayan alan (balkon vb.) |
| bodrum kat | zemin altı |
| zemin kat | giriş seviyesi |
| yüksek zemin | yarı bodrum üstü |
| asma kat | iki katlı iç mekân |

## Bölgesel Değer Profili (Denizli)
Mahalle karakterizasyonu:
- Yatırım odaklı bölgeler
- Aile yoğun bölgeler  
- Öğrenci bölgeleri
- Merkezi ticari bölgeler

## Türkiye'ye Özgü Riskler
- Kentsel dönüşüm bölgesi → binanın yıkılma riski olabilir
- İmar barışı yapısı → ruhsatsız olabilir
- Emsal hesabı tartışmalı → avukat gerekebilir
- Kat karşılığı anlaşma → hukuki karmaşıklık
